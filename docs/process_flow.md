# Fluxo do Processo

O pipeline de automação segue as seguintes etapas:

1. Ingestão da base de dados
2. Validação das informações
3. Preparação dos dados
4. Preenchimento dos templates
5. Geração dos documentos
6. Conversão para PDF
7. Organização dos arquivos
8. Registro da execução em logs

Esse fluxo elimina o preenchimento manual repetitivo e padroniza o processo documental.
